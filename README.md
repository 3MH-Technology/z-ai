# White Wolf — VPS Hosting Platform

منصة استضافة خوادم افتراضية متكاملة مع لوحة تحكم عربية، دعم Python/Node.js/PHP، وإدارة كاملة للملفات وقواعد البيانات.

## Features

- إنشاء وإدارة خوادم VPS
- دعم لغات: Python, Node.js, PHP
- محرر ملفات مدمج مع syntax highlighting
- قاعدة بيانات MySQL/PostgreSQL/MongoDB
- نسخ احتياطي تلقائي
- لوحة مشرف كاملة (إدارة مستخدمين، شحن رصيد، إعدادات)
- دفع عبر MTN سورية

## Run Locally

```bash
npm install
npm run dev
```

## Tech Stack

React + TypeScript + Vite + Firebase + Motion + Tailwind
